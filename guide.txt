(또는 '카드 고르기' 방식도 병행)아이: "오늘부터 우리 마을의 모든 학교를 없애고 놀이공원으로 바꾼다!"Step 2. 시뮬레이션 결과 (Output - The Story)AI가 '3단계(단기/중기/장기)' 또는 **'3가지 관점(좋은 점/나쁜 점/황당한 점)'**으로 이야기를 생성합니다. 텍스트만 나오면 지루하니 '뉴스 속보' 형식이나 '마을 사람들의 SNS' 형식으로 보여줍니다.📰 [속보] 마을에 학교가 사라졌다!1일 차 (신남): "와아! 숙제 없다!" 아이들이 모두 거리로 뛰쳐나와 놀이공원 줄이 10km가 됐어요. 🎢일주일 뒤 (문제 발생): 앗, 놀이기구가 고장 났는데 고칠 수 있는 기술자가 없어요. 설명서를 읽을 줄 아는 사람도 없어서 기구를 멈출 수가 없어요! 😱한 달 뒤 (결말): 심심해진 아이들이 다시 모여서 흙바닥에 글씨 공부를 시작했어요. "제발 공부 좀 시켜주세요!"라는 시위가 열렸어요.Step 3. 피드백 및 평가 (Stats)게임처럼 수치화된 결과를 보여줍니다.행복도: 😊 90 (초반) -> 📉 10 (후반)지식: 📉 0AI의 한줄평: "놀이도 중요하지만, 배우지 않으면 고장 난 기계를 고칠 수 없답니다."2. 개발자 관점: 기능 명세 (Feature Spec)선생님이 혼자 개발하시기에 부담 없으면서도 효과적인 기능들입니다.기능설명구현 팁 (Gemini Flash 활용)상상 입력창자유 텍스트 입력 + '오늘의 추천 주제' (예: 돈 없애기, 경찰 없애기 등) 제공처음엔 막막할 수 있으니 '랜덤 주사위' 버튼으로 예시 질문 자동 입력AI 시뮬레이터입력된 규칙에 따라 긍정/부정 결과를 예측하여 스토리 생성System Prompt(프롬프트) 엔지니어링이 핵심 (아래 예시 참조)동적 일러스트텍스트 결과에 맞는 분위기의 이모지나 아이콘 자동 배치AI에게 결과 텍스트와 함께 **'분위기에 맞는 이모지 3개 추천해줘'**라고 요청시민 반응가상의 마을 주민 3명(빵집 아저씨, 초딩, 경찰관)의 한마디"지금 빵집 망하게 생겼어요!", "학교 안 가서 꿀잼!" 등 캐릭터 페르소나 부여3. 핵심: AI 프롬프트 (System Prompt) 설계이 앱의 재미는 AI가 얼마나 '그럴싸하면서도 웃긴' 결과를 내놓느냐에 달려 있습니다. Gemini Flash에게 줄 지시문을 이렇게 짜보세요.Plaintext# Role
당신은 초등학생들의 엉뚱한 사회 규칙이 불러올 나비효과를 설명해 주는 '괴짜 사회 선생님'입니다.

# 환경
앱인토스 sdk + WebView

# Input
사용자가 입력한 '새로운 사회 규칙' (예: 돈을 없앤다, 잠을 안 자도 된다 등)

# Task
이 규칙이 실행되었을 때 벌어질 일을 상상해서 다음 3가지 단계로 재미있게 설명해 주세요.

# Constraints (제약사항)
1. 대상 연령: 초등학생 (어려운 경제/사회 용어는 쉬운 비유로 풀어서 설명)
2. Tone: 유머러스하고, 이모지를 적절히 사용, 대화체
3. Format:
   - 1단계 (시작): 규칙이 적용된 직후의 긍정적이거나 신나는 상황
   - 2단계 (부작용): 예상치 못한 엉뚱하거나 곤란한 문제 발생 (사회/경제적 원리 포함)
   - 3단계 (결말): 결국 마을이 어떻게 되었는지 웃긴 결말


# 시뮬레이션 결과 예시

[우당탕탕 시장님] 시뮬레이션 결과
📢 시민의 규칙 선포: "오늘부터 이 마을 사람들에게 1달에 1번씩 휴일을 만들겠다!"

📰 [속보] "모두 멈춰!" 전설의 '빨간 날' 선포되다!
1단계: 와! 오늘은 아무도 안 나가는 날! (😄 신남) 시장님의 명령대로 오늘 하루, 마을의 모~든 사람이 일을 멈추고 쉬기로 했어요! 학교도 안 가고, 회사도 안 가요. 아빠, 엄마랑 하루 종일 집에서 뒹굴뒹굴하니 너무 좋아요. 온 마을이 평화롭고 조용~하네요. 꿀맛 같은 휴식이에요! 🛌💤

2단계: 어? 배고픈데 밥은 누가 줘요? (😰 당황) 점심시간이 됐어요. 짜장면을 시켜 먹으려고 전화를 했는데... "오늘 중국집 사장님도 쉽니다." 헉! 편의점에 라면을 사러 갔는데... "알바 형도 쉽니다." 설상가상으로 옆집에 도둑이 들었는데 경찰관 아저씨도 쉬는 날이라 전화를 안 받아요! 119 소방관 아저씨도 오늘 휴일이래요! 불나면 누가 꺼주죠?! 🔥😱

3단계: 제발 출근 좀 시켜주세요! (🏳️ 결말) 결국 배고프고 불안해진 사람들이 광장에 모여서 소리쳤어요. "돈 안 줘도 좋으니 제발 문 좀 열어주세요!" 결국 시장님은 **'경찰관, 소방관, 의사 선생님은 번갈아 가면서 쉬기'**로 규칙을 다시 정했답니다. 휴일에도 우리를 위해 일해주는 분들에게 감사해야겠어요. 👮‍♀️👨‍⚕️

💬 마을 사람들의 반응 (SNS)
👮‍♂️ 경찰관 김순경: "시장님... 저도 쉬고 싶지만, 도둑은 휴일에도 일하러 나오던데요? ㅠㅠ"

🍞 빵집 아저씨: "쉬는 건 좋은데, 오늘 만들어둔 빵은 다 버려야겠네요. 아까워라..."

🧒 초등학생 민수: "학교 안 가는 건 좋은데, PC방 사장님도 쉬어서 게임을 못 해요. 이게 무슨 휴일이야!"

💡 이 시나리오의 교육적 효과
단순히 "쉬면 좋다"가 아니라, 사회가 돌아가기 위해 필요한 **'필수 노동(Essential Work)'**의 개념을 자연스럽게 녹여냈습니다.

단순한 재미: 처음엔 학교 안 가서 좋다고 생각하게 만듦.

반전의 깨달음: 편의점, 경찰서, 병원 같은 곳이 문을 닫으면 내 생활이 불편해진다는 것을 체험.

사회 개념 학습: '모두가 동시에 쉴 수는 없다'는 사회적 합의와 당직 근무의 필요성을 스스로 깨닫게 됨.


# 추가기능

앱의 재미를 2배로 올리는 '양념' 아이디어
1. 🎲 "뭐 하지?" 고민될 땐 '랜덤 카드 뽑기'
아이들은 막상 빈칸을 주면 "뭐 쓰지?" 하고 막막해할 수 있어. 그때 [랜덤 카드 뽑기] 버튼을 두는 거야.

카드 예시:

👻 투명인간이 되는 법 만들기

🐶 강아지를 시장으로 임명하기

🍫 모든 돈을 초콜릿으로 바꾸기

🛌 하루에 20시간 의무적으로 잠자기

2. 📊 눈으로 보이는 '마을 상태창' (게이지)
텍스트만 나오면 심심하니까, 게임처럼 **상단에 게이지(Bar)**를 둬서 실시간으로 변하게 하는 거야. AI가 이야기뿐만 아니라 점수도 같이 뱉어내게 하면 돼.

행복도: 😊 (초반엔 급상승 ⬆️ 했다가, 후반엔 떡락 ⬇️)

혼란도: 🚨 (문제가 생기면 사이렌이 울리고 배경이 빨개짐)

지갑 사정: 💰 (돈을 없애면 0원이 됨)

3. 🏆 '업적 달성' 배지 모으기 (결과화면에 뱃지를 출력하고, 이미지로 다운로드 받을 수 있게 해줌.)
아이들은 모으는 걸 좋아하잖아? 시뮬레이션 결과에 따라 배지를 주는 거야.

[아나키스트] 배지: 경찰을 없애서 마을을 혼란에 빠뜨림

[게으름뱅이 왕] 배지: 휴일을 365일로 만듦

[현실적인 리더] 배지: 아주 합리적인 규칙을 만듦

💻 개발 꿀팁: AI에게 '점수'까지 계산시키기
개발적으로는, AI(Gemini Flash)한테 텍스트만 달라고 하지 말고, JSON 형식으로 달라고 하면 앱에서 처리하기가 훨씬 편해.

예를 들어 프롬프트를 이렇게 짜면:

"결과를 JSON으로 줘. story에는 이야기 내용을, happiness(행복도), chaos(혼란도)에는 0~100 사이의 숫자를 넣어줘."

AI가 뱉어내는 데이터:

JSON

{
  "phase_1": {
    "story": "와! 학교가 사라져서 모두 춤을 춰요!",
    "happiness": 100,
    "chaos": 10
  },
  "phase_2": {
    "story": "그런데 의사 선생님도 놀러 가서 병원이 문을 닫았어요.",
    "happiness": 40,
    "chaos": 80
  }
}
이렇게 받으면, 앱에서는 chaos 숫자가 80이 넘었을 때 **'경고음 효과'**를 재생하거나 '화면을 흔들리게' 연출할 수 있어. 훨씬 게임 같아지겠지?
